## Installation
```
git clone git@github.com:jeggy/trialbee-test.git
cd trialbee-test
npm install
npm run sync-database
npm start
```
give it a minute or two to startup.<br>
Then just open [http://localhost:3000](http://localhost:3000 "Title")

* git
* npm
* node >= v6
